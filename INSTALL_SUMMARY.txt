================================================================================
                              TROUBLESHOOTING
================================================================================

Issue: "Node.js not found"
Solution: Install from nodejs.org/download

Issue: "Port 3001 already in use"
Solution: Kill existing process on port 3001
  Windows: netstat -ano | findstr :3001 → taskkill /PID <PID> /F
  Linux:   lsof -i :3001 → kill -9 <PID>

Issue: "Frontend shows blank page"
Solution:
  1. Verify backend is running
  2. Clear browser cache (Ctrl+Shift+Delete)
  3. Check browser console (F12) for errors
  4. Verify frontend/dist/ exists

Issue: "npm install fails"
Solution:
  cd frontend
  rm -rf node_modules package-lock.json
  npm install --legacy-peer-deps

Full troubleshooting guide: See INSTALL.md
